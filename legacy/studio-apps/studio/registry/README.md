# Component registry

Used by `ComponentPreview` to lazy-load example components in docs.

- **`default/example/*.tsx`** - Example components. Each file should export a single component (e.g. `ButtonDemo` in `button-demo.tsx`).
- **`__index__.tsx`** - Generated by `pnpm run build:registry`. Non-showcase examples only.

Showcase demos live in `packages/shared/.../showcase/registry/`. Run `pnpm run build:showcase-registry` from root to regenerate. It runs automatically before dev via root `dev:studio`/`dev:platform` and Studio `predev`. For non-showcase examples, add to `registry/default/example/` and run `pnpm run build:registry`.
