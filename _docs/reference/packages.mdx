# Package structure

## @forge/dev-kit

Re-exports shared + agent-engine. Install with pnpm:

```bash
pnpm add @forge/dev-kit
```

Components: EditorShell, EditorToolbar, EditorDockLayout, EditorDockPanel, EditorStatusBar, EditorInspector, EditorOverlaySurface.

## @forge/shared

Editor shell, DockLayout, Assistant domain contract, app/editor/viewport contracts.

## @forge/ui

Shadcn/Radix atoms. No domain deps.

## @forge/domain-*

Internal: domain-forge, domain-character. Not published.

## @forge/types

Payload-generated types. Internal.
