# Tutorial: AI integration

Add Assistant UI chat to your editor via `DomainAssistantContract`.

## Contract

```ts
interface DomainAssistantContract {
  domain: string;
  getContextSnapshot(): DomainContextSnapshot;
  getInstructions(): string;
  createTools(): DomainTool[];
  getSuggestions(): Array<{ title: string; message: string }>;
  onHighlight(entities: Record<string, string[]>): void;
  clearHighlights(): void;
}
```

## Steps

1. **Define contract** — Factory hook: `getContextSnapshot`, `getInstructions`, `createTools`, etc.
2. **Wire in editor** — `useDomainAssistant(contract, { enabled })` + `<DomainToolsRenderer contract={contract} />`
3. **Mount chat panel** — Render chat UI (e.g. `DialogueAssistantPanel` style) with `contract` and `toolsEnabled`
4. **Tools** — Each tool: `name`, `description`, `parameters`, `execute`, optional `render`
