# SettingsField

Individual settings field with auto-rendering, custom controls, settings store integration.

**File:** `apps/studio/components/settings/SettingsField.tsx`

## Overview

- **Auto-rendering** — Control type from field type
- **Custom controls** — Render prop for custom input
- **Store integration** — Binds to hierarchical settings store
- **Inheritance** — Shows override badge, reset

## Basic usage

```tsx
import { SettingsField } from '@/components/settings/SettingsField';
import { SettingsSection } from '@/components/settings/SettingsSection';

<SettingsSection scope="editor" scopeId="dialogue">
  <SettingsField
    fieldKey="ai.agentName"
    label="Agent Name"
    type="text"
    placeholder="Enter name"
  >
    <Input />
  </SettingsField>

  <SettingsField
    fieldKey="ai.autoSuggest"
    label="Auto Suggestions"
    type="toggle"
  >
    <Switch />
  </SettingsField>
</SettingsSection>
```

## Key props

| Prop | Description |
|------|-------------|
| `fieldKey` | Setting key (e.g. `ai.agentName`) |
| `label` | Field label |
| `type` | `text`, `toggle`, `select`, etc. |
| `description` | Help text |
| `default` | Default when not overridden |
