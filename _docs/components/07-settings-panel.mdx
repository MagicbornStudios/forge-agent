# SettingsPanel

Data-driven settings UI with scope inheritance, override badges, reset.

**File:** `apps/studio/components/settings/SettingsPanel.tsx`

## Overview

- **Hierarchical scopes** — app, project, editor, viewport
- **Inheritance** — Child scopes inherit from parent
- **Override visualization** — Badges when value overrides inherited
- **Reset** — Revert to inherited/default

## Basic usage

```tsx
import { SettingsPanel } from '@/components/settings/SettingsPanel';

const sections: SettingsSection[] = [
  {
    id: 'ai-core',
    title: 'AI Core',
    fields: [
      { key: 'ai.agentName', label: 'Agent Name', type: 'text' },
      { key: 'ai.autoSuggest', label: 'Auto Suggestions', type: 'toggle' },
    ],
  },
];

<SettingsPanel
  scope="editor"
  sections={sections}
  editorId="dialogue"
/>
```

## Props

| Prop | Description |
|------|-------------|
| `scope` | `'app' \| 'project' \| 'editor' \| 'viewport'` |
| `sections` | Section definitions with fields |
| `editorId` | For editor scope |
| `viewportId` | For viewport scope |
