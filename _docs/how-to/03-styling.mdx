# 03 - Styling and theming

Design tokens, editor themes, and where UI atoms live.

## Design tokens

Semantic tokens in `packages/shared/src/shared/styles/themes.css`:
`--background`, `--foreground`, `--border`, `--muted`, etc. Prefer these over raw colors.

## Theme

- **App theme** — `ui.theme` (settings) + `AppThemeProvider`; sets `<html data-theme="...">`. File: `apps/studio/components/providers/AppThemeProvider.tsx`
- **Editor override** — `EditorShell` accepts `theme`; sets `data-theme` on editor root

## UI atoms

From `packages/ui`: `@forge/ui/button`, `@forge/ui/card`, `@forge/ui/dialog`, etc. Add via `npx shadcn@latest add <name>` in the UI package.
