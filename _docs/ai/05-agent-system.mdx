# 05 - Agent system

BaseAgent, AgentRegistry, AppAgent with domain delegation.

## Flow

AppAgent receives request; delegates by scope (forge → ForgeAgent, character → CharacterAgent). Domain agents execute tools; AppAgent returns result.

## BaseAgent

- `AgentConfig`: id, name, description, instructions, tools, scopes
- `canHandleScope(scope)`
- `execute(context)` → AgentResult

## AgentRegistry

- `register(agent)`, `get(id)`, `findForScope(scope)`
- Used by LangGraph orchestrator when enabled
