# Tutorial: Adding panels to your editor

Multi-panel layout: navigator, main canvas, inspector, output.

## Step 1: Basic 2-pane

```tsx
import { EditorShell, EditorToolbar, EditorInspector, EditorStatusBar } from '@forge/shared';

<EditorShell>
  <EditorToolbar>...</EditorToolbar>
  <div className="flex flex-1 min-h-0">
    <div className="flex-1">Canvas</div>
    <EditorInspector selection={selection}>...</EditorInspector>
  </div>
  <EditorStatusBar>Ready</EditorStatusBar>
</EditorShell>
```

## Step 2: WorkspaceLayout

Use `WorkspaceLayout` with declarative slot children:

```tsx
<WorkspaceLayout>
  <WorkspaceLayout.Left>
    <WorkspaceLayout.Panel id="navigator" title="Navigator">
      <NavigatorPanel />
    </WorkspaceLayout.Panel>
  </WorkspaceLayout.Left>
  <WorkspaceLayout.Main>
    <WorkspaceLayout.Panel id="main" title="Main">
      <MainCanvas />
    </WorkspaceLayout.Panel>
  </WorkspaceLayout.Main>
  <WorkspaceLayout.Right>
    <WorkspaceLayout.Panel id="inspector" title="Inspector">
      <InspectorPanel />
    </WorkspaceLayout.Panel>
  </WorkspaceLayout.Right>
</WorkspaceLayout>
```

Slot children are canonical. Define panel placement directly in JSX via `WorkspaceLayout.Left/Main/Right/Bottom`.

For workspace apps, use one `WorkspaceLayout` per workspace tab and compose only the panels for that workspace.
