## What is Forge?

Forge is a production-ready platform for building professional desktop-class editors in the browser. ### Create Your First Editor

```typescript
import {
  EditorShell,
  EditorToolbar,
  EditorDockLayout,
  EditorStatusBar,
} from '@forge/shared/components/editor';

export function MyEditor() {
  return (
    <EditorShell editorId="my-editor" title="My Editor">
      <EditorToolbar>
        <EditorToolbar.Left>
          <Button>Action</Button>
        </EditorToolbar.Left>
      </EditorToolbar>

      <EditorDockLayout layoutId="my-editor-layout">
        <EditorDockLayout.Left>
          <EditorDockPanel id="navigator" title="Navigator">
            <MyNavigator />
          </EditorDockPanel>
        </EditorDockLayout.Left>

        <EditorDockLayout.Main>
          <EditorDockPanel id="viewport" title="Main">
            <MyViewport />
          </EditorDockPanel>
        </EditorDockLayout.Main>
      </EditorDockLayout>

      <EditorStatusBar>Ready</EditorStatusBar>
    </EditorShell>
  );
}
```
