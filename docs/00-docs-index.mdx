### 3. Contributor workspace requirement

We expect every new contributor to **build their own workspace** (tutorial: [05 - Building a workspace](how-to/05-building-a-workspace.mdx)) and **submit a PR** to add it under **`packages/shared/contributor_workspaces/`**. See the [README](../README.md) contributor section and [contributor_workspaces README](../packages/shared/contributor_workspaces/README.md).
