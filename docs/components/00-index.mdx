# Components

Editor primitives from `@forge/shared` and Studio settings. Build professional editors with EditorShell, EditorDockLayout, EditorDockPanel, and more.

## Showcase

Browse all components in one place. Navigate by tier:

| Tier | Description |
|------|-------------|
| [Atoms](components/showcase/atoms.mdx) | `@forge/ui` — Button, Input, Card, Badge, and 50+ primitives |
| [Molecules](components/showcase/molecules.mdx) | `@forge/shared` — EditorShell, DockLayout, EditorToolbar |
| [Organisms](components/showcase/organisms.mdx) | Full editors — DialogueEditor, CharacterEditor, Studio |

## Quick start

```tsx
import {
  EditorShell,
  EditorToolbar,
  EditorDockLayout,
  EditorDockPanel,
  EditorStatusBar,
} from '@forge/shared/components/editor';

<EditorShell editorId="my-editor" title="My Editor">
  <EditorToolbar>...</EditorToolbar>
  <EditorDockLayout layoutId="my-layout">
    <EditorDockLayout.Main>
      <EditorDockPanel panelId="main" title="Viewport">
        <MyContent />
      </EditorDockPanel>
    </EditorDockLayout.Main>
  </EditorDockLayout>
  <EditorStatusBar>Ready</EditorStatusBar>
</EditorShell>
```
