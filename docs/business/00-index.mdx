## Summary

- **First revenue** = paid clone: buyers pay the price the creator sets to clone a project (or template, or Strategy core). Listing creation is free.
- **Stripe Connect from day one** so creators get paid directly; platform takes a cut. We use Stripe hosted Checkout and Stripe for invoicingâ€”no custom payment forms.
- **Clone** = full project copy (settings, characters, graphs, pages, etc.) with media as references until the clone owner replaces them. A **license** record enables "clone again."
- **Developer program** (after MVP): developers can submit editors/apps (web tech, dev kit); approved editors receive a revenue split while in the official suite; community apps are listable and installable without payouts. See [Developer program and editor ecosystem](developer-program-and-editors.mdx).
- **Ecosystem adapters:** Import from Yarn Spinner (first), then RenPy, Ink, Twine; Pixi'VN as universal runtime; we generate assets so creators get renewed interest. See [Ecosystem and import strategy](ecosystem-and-import-strategy.mdx).
