# 02 - Component library + Verdaccio

Publishable blocks: @forge/ui, @forge/shared, @forge/agent-engine, @forge/dev-kit.

## Published packages

| Package | Contents |
|---------|----------|
| @forge/ui | Shadcn/Radix atoms, no domain deps |
| @forge/shared | Editor shell, DockLayout, Assistant helpers, domain contracts |
| @forge/agent-engine | Workflow runtime (steps, events, SSE) |
| @forge/dev-kit | Re-exports shared + agent-engine |

**Not published:** @forge/domain-* (internal), @forge/types.

## Verdaccio

```bash
pnpm registry:start
```

Config: `verdaccio.yaml`. See [how-to/25-verdaccio](../how-to/25-verdaccio-local-registry).

## Build + publish

```bash
pnpm registry:forge:build
pnpm registry:forge:publish:local
```
