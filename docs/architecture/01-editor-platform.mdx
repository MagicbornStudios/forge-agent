# 01 - Editor platform

Unreal-style dockable panels. WorkspaceApp, WorkspaceShell, DockLayout.

## Naming

| Old | New | Import |
|-----|-----|--------|
| AppSpace | WorkspaceApp | @forge/shared/components/app |
| WorkspaceShell | WorkspaceShell | @forge/shared/components/workspace |
| WorkspaceLayoutGrid | DockLayout | @forge/shared/components/workspace |
| WorkspaceLeftPanel | DockPanel | @forge/shared/components/workspace |

## Hierarchy

```
WorkspaceApp
  WorkspaceApp.Tabs
  WorkspaceApp.Content
    WorkspaceShell (editorId, domain, theme)
      WorkspaceHeader
      WorkspaceToolbar
      WorkspaceReviewBar
      DockLayout (left, main, right, bottom)
        DockPanel
          PanelTabs
          PanelSettings
        ViewportMeta
      WorkspaceStatusBar
      WorkspaceOverlaySurface
```

## WorkspaceShell

```tsx
<WorkspaceShell editorId="dialogue" title="Dialogue" domain="dialogue" theme="dark-fantasy">
  {/* slots */}
</WorkspaceShell>
```

Sets `data-editor-id`, `data-domain`, `data-theme`, `data-density`.

## DockLayout

Dockview-based: drag-to-reorder, floating groups, tabbed panels.
