# Prerequisites

Before you start, make sure you have:

- **Node 20+** — Run `node -v` to check
- **pnpm** — Run `pnpm -v` to check. This repo pins pnpm via `packageManager`; use Corepack to match CI:
  ```bash
  corepack enable
  corepack prepare pnpm@9.15.5 --activate
  ```
  Or: `corepack enable` (Node will use the version from `package.json`).

## Optional but useful

- **Git** — For cloning and contributing
- **A code editor** — Cursor, VS Code, or similar

That's it. Next: install and env.
