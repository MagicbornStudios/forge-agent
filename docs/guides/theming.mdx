# Theming

Design tokens, `data-theme`, `data-density`. See [design/01-styling-and-theming](../design/01-styling-and-theming).

## Tokens

- `--control-height`, `--control-gap`, `--panel-padding`, `--tab-height`
- `--background`, `--foreground`, `--border`, `--radius-*`

## Density

- Default: `compact`
- Override: `EditorShell` sets `data-density`; `[data-density='comfortable']` for relaxed

## Themes

`data-theme="light"` or `data-theme="dark"`. Base from shadcn; one primary accent.
