# TypeScript interfaces

## Selection

Discriminated union: `none`, `entity`, `textRange`, `canvasObject`.

```ts
type Selection =
  | { type: 'none' }
  | { type: 'entity'; entityType: string; id: string; meta?: Record<string, unknown> }
  | SelectionTextRange
  | SelectionCanvasObject;
```
