# 02 - Editor shell and slots

Every editor uses the same surfaces. Composition only; no data binding in this guide.

## Shared components

From `packages/shared/src/shared/components/workspace/`:

| Component | Purpose |
|-----------|---------|
| WorkspaceShell | Root; `editorId`, `title`, `domain`, `theme`, `density` |
| WorkspaceToolbar | `.Left`, `.Right`, `.Center`, `.Group`, `.Button`, `.Menubar` |
| WorkspaceLayout | Slots: `.Left`, `.Main`, `.Right`, `.Bottom` |
| WorkspacePanel | Panel wrapper; title, tabs, scrollable |
| WorkspaceStatusBar | Footer strip |
| WorkspaceOverlaySurface | `overlays`, `activeOverlay`, `onDismiss` |

## Minimal shell

```tsx
<WorkspaceShell editorId="my-editor" title="My Editor" domain="my">
  <WorkspaceToolbar>
    <WorkspaceToolbar.Left><span>My Editor</span></WorkspaceToolbar.Left>
  </WorkspaceToolbar>
  <WorkspaceLayout layoutId="my-layout">
    <WorkspaceLayout.Main>
      <WorkspacePanel panelId="main" title="Main">
        Main content
      </WorkspacePanel>
    </WorkspaceLayout.Main>
  </WorkspaceLayout>
  <WorkspaceStatusBar>Ready</WorkspaceStatusBar>
  <WorkspaceOverlaySurface overlays={[]} activeOverlay={null} onDismiss={() => {}} />
</WorkspaceShell>
```

Reference: `apps/studio/components/workspaces/DialogueWorkspace.tsx`, `apps/studio/components/workspaces/CharacterWorkspace.tsx`.
