# 02 - Editor shell and slots

Every editor uses the same surfaces. Composition only; no data binding in this guide.

## Shared components

From `packages/shared/src/shared/components/editor/`:

| Component | Purpose |
|-----------|---------|
| EditorShell | Root; `editorId`, `title`, `domain`, `theme`, `density` |
| EditorToolbar | `.Left`, `.Right`, `.Center`, `.Group`, `.Button`, `.Menubar` |
| WorkspaceLayout | Slots: `.Left`, `.Main`, `.Right`, `.Bottom` |
| WorkspacePanel | Panel wrapper; title, tabs, scrollable |
| EditorStatusBar | Footer strip |
| EditorOverlaySurface | `overlays`, `activeOverlay`, `onDismiss` |

## Minimal shell

```tsx
<EditorShell editorId="my-editor" title="My Editor" domain="my">
  <EditorToolbar>
    <EditorToolbar.Left><span>My Editor</span></EditorToolbar.Left>
  </EditorToolbar>
  <WorkspaceLayout layoutId="my-layout">
    <WorkspaceLayout.Main>
      <WorkspacePanel panelId="main" title="Main">
        Main content
      </WorkspacePanel>
    </WorkspaceLayout.Main>
  </WorkspaceLayout>
  <EditorStatusBar>Ready</EditorStatusBar>
  <EditorOverlaySurface overlays={[]} activeOverlay={null} onDismiss={() => {}} />
</EditorShell>
```

Reference: `apps/studio/components/workspaces/DialogueWorkspace.tsx`, `apps/studio/components/workspaces/CharacterWorkspace.tsx`.
