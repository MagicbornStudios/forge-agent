# 01 - Architecture overview

Package structure and flow for the Assistant UI AI system.

## Flow

1. User types in Assistant chat panel
2. Chat → POST /api/assistant-chat (AI SDK streaming)
3. Model may call domain tools (`forge_createNode`, `character_create`)
4. Tools execute; results render (Plan, Progress Tracker)
5. Editor applies changes, highlights entities

## Packages

- **@forge/assistant-runtime** — OpenRouter adapter, streaming, agents, tools
- **packages/shared/assistant/** — DomainAssistantContract, use-domain-assistant
- **packages/domain-forge** — Forge contract and tools
- **packages/domain-character** — Character contract and tools
