# Component registry

Used by `ComponentPreview` to lazy-load example components in docs.

- **`default/example/*.tsx`** – Example components. Each file should export a single component (e.g. `ButtonDemo` in `button-demo.tsx`).
- **`__index__.tsx`** – Generated by `pnpm run build:registry`. Do not edit by hand.

Run `pnpm run build:registry` after adding or renaming examples. It runs automatically before `pnpm dev` via `predev`.
