## Package Architecture

Forge provides a single, unified package for building professional editors:

## Installation

```bash
npm install @forge/dev-kit
```

The dev-kit includes everything you need:
- Editor platform components (EditorShell, DockLayout, etc.)
- UI components (Button, Card, Dialog, etc.)
- AI integration utilities
- TypeScript types
- Settings system
```

## What's in @forge/dev-kit

### Editor Components

```typescript
// Core editor components
import {
  EditorShell,
  EditorToolbar,
  WorkspaceLayout,
  WorkspacePanel,
  EditorStatusBar,
  EditorInspector,
  EditorOverlaySurface,
} from '@forge/dev-kit';
```

**Available Components:**
- EditorShell - Root container with theming
- EditorToolbar - Customizable toolbar
- WorkspaceLayout - VSCode-style panels
- WorkspacePanel - Individual panel wrapper
- EditorStatusBar - Bottom status bar
- EditorInspector - Properties panel
- EditorOverlaySurface - Modal system

---

## Import Patterns

### Editor Components

```typescript
// Import editor components
import {
  EditorShell,
  WorkspaceLayout,
  WorkspacePanel,
  EditorToolbar,
  EditorStatusBar,
} from '@forge/dev-kit';
```

### UI Components

```typescript
// Import UI primitives

  Button,

  Badge,

  Tooltip,

```typescript
// Import types
import type {

## Peer Dependencies

@forge/dev-kit requires:

```json
{
  "peerDependencies": {
    "react": "^19.0.0",
    "react-dom": "^19.0.0"
  }
}
```

Install these alongside dev-kit:

```bash
npm install react@^19.0.0 react-dom@^19.0.0 @forge/dev-kit
```

### Quick Start

```typescript
import { EditorShell, WorkspaceLayout } from '@forge/dev-kit';
import { Button, Card } from '@forge/dev-kit/ui';

export function MyEditor() {
  return (
    <EditorShell editorId="my-editor" title="My Editor">
      {/* Your editor content */}
    </EditorShell>
  );
}
```
